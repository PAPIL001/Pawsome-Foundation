# Pawsome Foundation

A full-stack React application for the Pawsome Foundation animal rescue and sanctuary in Lucknow, India.

Live link: [https://pawsome-foundation.vercel.app/](https://pawsome-foundation.vercel.app/)

## Tech Stack
* **Frontend:** React, Vite, Tailwind CSS, Framer Motion
* **Backend:** Node.js, Express (MVC architecture)
* **Design:** Mobile-first, Dark-mode supported, Glassmorphism elements

## Project Structure
* `client/` - React frontend
* `server/` - Express backend API

## Setup Instructions

### Frontend (Client)
1. Navigate to the `client` directory: `cd client`
2. Install dependencies: `npm install`
3. Run dev server: `npm run dev` (runs on http://localhost:5173)

### Backend (Server)
1. Navigate to the `server` directory: `cd server`
2. Install dependencies: `npm install`
3. Start server: `node server.js` (runs on port 5000)

## Features
* **Animated Counters:** Showcasing the real impact (rescues, meals, lives saved).
* **Interactive Gallery:** Masonry grid with category filters and a custom media lightbox supporting local images and MP4 videos.
* **Donation Portal:** Easy-access bank details and QR code for direct contributions.
* **Responsive Design:** Optimized for all screen sizes with a custom mobile navigation drawer.
